FieldTypeInfo
schema string required
Configuration schema version. Current examples use v0
bootloader string required
Boot loader: grub or none
kernelCmdLine string optional
Extra kernel boot parameters appended after essential params
serialConsole bool optional
Enable serial console output for the bootloader
cryptoPolicy string optional
Crypto policy: fips or default. When fips, appends FIPS kernel cmdline args
raw.diskSize DiskSize required*
Size of the raw disk image.
Pattern: ^[1-9]\d*[KMGT]$
Defaults: 10G (build) / 12G (customize)
iso.device string required†
Target install disk for ISO mode, e.g. /dev/sda

* Required when image type is raw if you do not want the command default. Falls back to 10G for elemental3 build and 12G for elemental3 customize --type raw.
† Required when image type is iso. Mutually exclusive with raw.
